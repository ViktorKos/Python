# Реалізація проекту Django

Для роботи проекта необхідний файл `/notes/.env` зі змінними оточення.
Створіть його з таким вмістом і підставте свої значення.

```dotenv
SECRET_KEY=

DATABASE_NAME=
DATABASE_USER=
DATABASE_PASSWORD=
DATABASE_HOST=
DATABASE_PORT=

EMAIL_HOST=
EMAIL_PORT=
EMAIL_HOST_USER=
EMAIL_HOST_PASSWORD=
```

Запуск застосунку

```
cd notes
py manage.py runserver
```
